# Nmap Network Discovery Report

## Introduction

Network discovery is the process of identifying devices connected to a network. Nmap (Network Mapper) is one of the most widely used open-source tools for network scanning and security auditing.

---

## Objectives

- Discover active hosts on a network.
- Identify open ports.
- Detect running services.
- Learn basic Nmap scanning techniques.
- Document scan results.

---

## Tools Used

- Nmap
- Kali Linux
- Visual Studio Code
- Git
- GitHub

---

## Methodology

The following Nmap scan types are included in this project:

1. Ping Scan
2. Service Version Detection
3. Operating System Detection
4. Aggressive Scan

---

## Expected Results

The scan should identify:

- Active hosts
- Open TCP ports
- Running services
- Operating system information

---

## Recommendations

- Disable unused services.
- Close unnecessary ports.
- Enable firewall protection.
- Keep systems updated.
- Use strong authentication.

---

## Conclusion

Nmap is a powerful network discovery and security auditing tool. It helps administrators understand their network and identify potential security issues.
