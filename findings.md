# Scan Findings

## Summary

This project demonstrates the use of Nmap for basic network discovery.

### Information Collected

- Active hosts
- Open ports
- Running services
- Service versions
- Operating system information

## Security Recommendations

- Close unnecessary ports.
- Disable unused services.
- Keep systems updated.
- Enable firewall protection.
- Use strong passwords.

## Conclusion

Nmap is an essential tool for network administrators and cybersecurity professionals. It helps identify network devices and possible security risks.
