# Nmap Network Discovery Project

## Project Overview

This project demonstrates how to use Nmap for network discovery in a controlled lab environment. It identifies active hosts, open ports, running services, and operating system information.

## Objectives

- Discover live hosts on a network.
- Identify open TCP ports.
- Detect running services and versions.
- Perform basic operating system detection.
- Document the results.

## Tools Used

- Kali Linux
- Nmap
- Visual Studio Code
- Git
- GitHub
